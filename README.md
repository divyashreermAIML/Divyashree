# 🚦 Smart Traffic Control System Using AI-Based Dynamic Signal Timing

## 📌 Project Overview

The Smart Traffic Control System is an AI-based traffic management solution that dynamically adjusts traffic signal timings according to real-time traffic density. The system reduces congestion, minimizes vehicle waiting time, and provides priority to emergency vehicles.

## 🎯 Problem Statement

Traditional traffic signals use fixed timing regardless of traffic conditions, causing unnecessary delays and congestion. This project solves this problem using intelligent dynamic signal timing.

## ✨ Features

- 🚗 AI-based dynamic traffic signal control
- 🚑 Emergency vehicle priority
- 📊 Real-time traffic monitoring dashboard
- 📈 Traffic analysis and visualization
- 🚦 Fixed-Time vs Smart-Time comparison
- 🌐 2×2 grid traffic simulation

## 🛠 Technologies Used

- Python
- Streamlit
- Pandas
- Matplotlib
- SUMO (Simulation of Urban Mobility)

